```
GET /api/sites/:site/session-replay/:sessionId
```

Returns the rrweb events and metadata needed to reconstruct a session replay.

### Path Parameters

<TypeTable
  type={{
    site: {
      description: 'Site ID or site identifier',
      type: 'string | number',
      required: true,
    },
    sessionId: {
      description: 'The session to fetch',
      type: 'string',
      required: true,
    },
  }}
/>

### Query Parameters

This endpoint takes no query parameters.

### Response

<TypeTable
  type={{
    events: {
      description: 'Array of rrweb event objects, replayed in order to reconstruct the session. Each carries a timestamp and rrweb-format payload.',
      type: 'RRWebEvent[]',
    },
    metadata: {
      description: "The session replay metadata object (same fields as a list item, enriched with the user's traits)",
      type: 'SessionReplayMetadata',
    },
  }}
/>
