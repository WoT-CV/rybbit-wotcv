```
GET /api/sites/:site/session-replay/list
```

Returns recorded session replays with metadata, newest first.

### Path Parameters

<TypeTable
  type={{
    site: {
      description: 'Site ID or site identifier',
      type: 'string | number',
      required: true,
    },
  }}
/>

### Query Parameters

Accepts all [Common Parameters](/docs/api/getting-started#common-parameters) (time and filters) plus the following:

<TypeTable
  type={{
    limit: {
      description: 'Number of results to return',
      type: 'number',
      default: 50,
    },
    offset: {
      description: 'Number of results to skip',
      type: 'number',
      default: 0,
    },
    userId: {
      description: 'Restrict to a single user',
      type: 'string',
      required: false,
    },
    minDuration: {
      description: 'Minimum replay duration in seconds',
      type: 'number',
      required: false,
    },
  }}
/>

### Response

<TypeTable
  type={{
    data: {
      description: 'Array of session replay objects',
      type: 'SessionReplay[]',
    },
  }}
/>
