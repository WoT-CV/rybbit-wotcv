```
GET /api/organizations/:organizationId/sites
```

Returns all sites in an organization the caller can access, with usage and subscription info.

### Path Parameters

<TypeTable
  type={{
    organizationId: {
      description: 'Organization ID',
      type: 'string',
      required: true,
    },
  }}
/>

### Query Parameters

This endpoint takes no query parameters.

### Response

<TypeTable
  type={{
    organization: {
      description: 'The organization record, or null',
      type: 'object | null',
    },
    sites: {
      description: 'Array of site objects',
      type: 'array',
    },
    subscription: {
      description: 'Usage and subscription details for the organization',
      type: 'object',
    },
  }}
/>

#### Site Object

Each site in the `sites` array contains the following fields. Other site-config fields may also be present:

<TypeTable
  type={{
    siteId: {
      description: 'Site ID',
      type: 'number',
    },
    name: {
      description: 'Site display name',
      type: 'string',
    },
    domain: {
      description: 'Site domain',
      type: 'string',
    },
    type: {
      description: 'Site type',
      type: 'string',
      typeDescription: 'e.g. "web"',
    },
    public: {
      description: 'Whether the site dashboard is public',
      type: 'boolean',
    },
    organizationId: {
      description: 'Organization ID the site belongs to',
      type: 'string',
    },
    sessionsLast24Hours: {
      description: 'Number of sessions in the last 24 hours',
      type: 'number',
    },
    isOwner: {
      description: 'Whether the caller owns the site',
      type: 'boolean',
    },
    teams: {
      description: 'Teams the site belongs to',
      type: 'array',
      typeDescription: 'Array of { id, name }',
    },
  }}
/>
