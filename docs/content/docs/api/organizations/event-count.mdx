```
GET /api/org-event-count/:organizationId
```

Returns daily event usage counts across an organization's sites.

### Path Parameters

<TypeTable
  type={{
    organizationId: {
      description: 'Organization ID',
      type: 'string',
      required: true,
    },
  }}
/>

### Query Parameters

<TypeTable
  type={{
    start_date: {
      description: 'Start of the date range. Omit start/end to get all-time daily counts.',
      type: 'string',
      typeDescription: 'YYYY-MM-DD',
    },
    end_date: {
      description: 'End of the date range. Omit start/end to get all-time daily counts.',
      type: 'string',
      typeDescription: 'YYYY-MM-DD',
    },
    time_zone: {
      description: 'IANA timezone used to bucket events into days',
      type: 'string',
      default: 'UTC',
    },
  }}
/>

<TypeTable
