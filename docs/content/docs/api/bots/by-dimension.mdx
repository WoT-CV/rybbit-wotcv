```
GET /api/sites/:site/bots/by-dimension
```

Returns bot requests broken down by a dimension.

### Path Parameters

<TypeTable
  type={{
    site: {
      description: 'Site ID or site identifier',
      type: 'string | number',
      required: true,
    },
  }}
/>

### Query Parameters

Accepts all [Common Parameters](/docs/api/getting-started#common-parameters) (time and filters).

Bot filtering supports a subset of filter parameters: `browser`, `browser_version`, `operating_system`, `operating_system_version`, `country`, `region`, `city`, `device_type`, `referrer`, `hostname`, `pathname`, `querystring`, `dimensions`, `user_id`, `lat`, and `lon`.

<TypeTable
  type={{
    dimension: {
      description: 'The dimension to group by',
      type: 'string',
      typeDescription: '"browser" | "browser_version" | "operating_system" | "operating_system_version" | "country" | "region" | "city" | "device_type" | "referrer" | "hostname" | "pathname" | "dimensions" | "asn_org" | "bot_category" | "matched_ua_pattern"',
      required: true,
    },
    limit: {
      description: 'Number of results per page',
      type: 'number',
      default: 100,
    },
    page: {
      description: 'Page number (1-indexed)',
      type: 'number',
      default: 1,
    },
    layer: {
      description: 'Restrict results to a single detection layer',
      type: 'string',
      typeDescription: '"ua_pattern" | "header_heuristics" | "client_signals" | "bot_asn" | "rate_anomaly"',
      required: false,
    },
  }}
/>
