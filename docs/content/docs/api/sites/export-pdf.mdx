```
GET /api/sites/:site/export/pdf
```

Generates a downloadable PDF analytics report for a date range.

### Path Parameters

<TypeTable
  type={{
    site: {
      description: 'Site ID or site identifier',
      type: 'string | number',
      required: true,
    },
  }}
/>

### Query Parameters

<TypeTable
  type={{
    start_date: {
      description: 'Start of the date range',
      type: 'string',
      typeDescription: 'YYYY-MM-DD',
      required: true,
    },
    end_date: {
      description: 'End of the date range',
      type: 'string',
      typeDescription: 'YYYY-MM-DD',
      required: true,
    },
    time_zone: {
      description: 'IANA timezone',
      type: 'string',
      required: true,
    },
    filters: {
      description: 'JSON-encoded filter array (same format as the Common Parameters filters)',
      type: 'string',
    },
  }}
/>

### Response

Returns a binary PDF file (`Content-Type: application/pdf`). Save the response body to a `.pdf` file rather than parsing it as JSON.
