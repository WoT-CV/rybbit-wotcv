```
GET /api/sites/:site/events/count
```

Returns time-bucketed event counts grouped by event type.

### Path Parameters

<TypeTable
  type={{
    site: {
      description: 'Site ID or site identifier',
      type: 'string | number',
      required: true,
    },
  }}
/>

### Query Parameters

Accepts all [Common Parameters](/docs/api/getting-started#common-parameters) (time and filters).

<TypeTable
  type={{
    bucket: {
      description: 'Time bucket size for grouping data',
      type: 'TimeBucket',
      typeDescription: '"minute" | "five_minutes" | "ten_minutes" | "fifteen_minutes" | "hour" | "day" | "week" | "month" | "year"',
      default: 'day',
    },
  }}
/>

### Response

<TypeTable
  type={{
    time: {
      description: 'Start of the time bucket',
      type: 'string',
    },
    pageview_count: {
      description: 'Number of pageviews in this bucket',

    },
    custom_event_count: {
      description: 'Number of custom events in this bucket',
      type: 'number',
    },
    performance_count: {
      description: 'Number of performance events in this bucket',
      type: 'number',

    outbound_count: {
      description: 'Number of outbound link events in this bucket',

      description: 'Number of error events in this bucket',

    button_click_count: {
      description: 'Number of button click events in this bucket',

      description: 'Number of copy events in this bucket',

    },
  }}
/>
