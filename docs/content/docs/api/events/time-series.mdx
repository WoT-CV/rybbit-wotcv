```
GET /api/sites/:site/events/time-series
```

Returns time-bucketed counts for the top custom events.

### Path Parameters

<TypeTable
  type={{
    site: {
      description: 'Site ID or site identifier',
      type: 'string | number',
      required: true,
    },
  }}
/>

### Query Parameters

Accepts all [Common Parameters](/docs/api/getting-started#common-parameters) (time and filters).

<TypeTable
  type={{
    bucket: {
      description: 'Time bucket size for grouping data',
      type: 'TimeBucket',
      typeDescription: '"minute" | "five_minutes" | "ten_minutes" | "fifteen_minutes" | "hour" | "day" | "week" | "month" | "year"',
      default: 'hour',
    },
    limit: {
      description: 'Number of top custom events (by total count) to include',
      type: 'number',
      typeDescription: 'Range: 1-10',
      default: 5,
    },
  }}
/>

### Response

<TypeTable
  type={{
    time: {
      description: 'Start of the time bucket',
      type: 'string',
    },
    event_name: {
      description: 'Name of the custom event',
      type: 'string',
    },
    event_count: {
      description: 'Number of times the event occurred in this bucket',
      type: 'number',
    },
  }}
/>
