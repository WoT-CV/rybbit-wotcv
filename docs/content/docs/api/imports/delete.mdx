```
DELETE /api/sites/:site/imports/:importId
```

Deletes an import and all events it added. The import must be finished. Requires an admin or owner API key. On Rybbit Cloud, this requires a paid plan.

A still-running import cannot be deleted and returns HTTP `400`.

### Path Parameters

<TypeTable
  type={{
    site: {
      description: 'Site ID or site identifier',
      type: 'string | number',
      required: true,
    },
    importId: {
      description: 'UUID of the import to delete',
      type: 'string',
      required: true,
    },
  }}
/>

### Response

Returns `200 OK` with an empty body.
