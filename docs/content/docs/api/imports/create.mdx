```
POST /api/sites/:site/imports
```

Starts a new data import from a supported platform. Requires an admin or owner API key. On Rybbit Cloud, this requires a paid plan.

Only one concurrent import is allowed per organization. Starting a second import while one is still running returns HTTP `429`.

### Path Parameters

<TypeTable
  type={{
    site: {
      description: 'Site ID or site identifier',
      type: 'string | number',
      required: true,
    },
  }}
/>

### Request Body

<TypeTable
  type={{
    platform: {
      description: 'Source platform to import data from',
      type: 'string',
      typeDescription: '"umami" | "simple_analytics" | "plausible"',
      required: true,
    },
  }}
/>

### Response

<TypeTable
  type={{
    data: {
      description: 'Import details',
      type: 'CreateImportData',
    },
  }}
/>

#### CreateImportData Object

<TypeTable
  type={{
    importId: {
      description: 'UUID of the created import. Use this when uploading events.',
      type: 'string',
      typeDescription: 'UUID',
    },
    allowedDateRange: {
      description: 'The inclusive date window you may import events within, based on your plan\'s retention',
      type: 'AllowedDateRange',
    },
  }}
/>

#### AllowedDateRange Object

<TypeTable
  type={{
    earliestAllowedDate: {
      description: 'Earliest date you may import events for (inclusive)',

      typeDescription: 'YYYY-MM-DD',
    },
    latestAllowedDate: {
      description: 'Latest date you may import events for (inclusive)',
      type: 'string',
      typeDescription: 'YYYY-MM-DD',
    },
  }}
/>
