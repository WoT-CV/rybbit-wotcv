```
GET /api/sites/:site/imports
```

Returns all data imports for a site with their progress counts. Requires an admin or owner API key. On Rybbit Cloud, this requires a paid plan.

### Path Parameters

<TypeTable
  type={{
    site: {
      description: 'Site ID or site identifier',
      type: 'string | number',
      required: true,
    },
  }}
/>

### Query Parameters

This endpoint takes no query parameters.

### Response

<TypeTable
  type={{
    data: {
      description: 'Array of import objects',
      type: 'Import[]',
    },
  }}
/>

#### Import Object

<TypeTable
  type={{
    importId: {
      description: 'Unique import identifier',
      type: 'string',
      typeDescription: 'UUID',
    },
    platform: {
      description: 'Source platform the data was imported from',
      type: 'string',
      typeDescription: '"umami" | "simple_analytics" | "plausible"',
    },
    importedEvents: {
      description: 'Number of events successfully imported',
      type: 'number',
    },
    skippedEvents: {
      description: 'Number of events skipped (e.g. outside the retention window)',
      type: 'number',
    },
    invalidEvents: {
      description: 'Number of events rejected as invalid',
      type: 'number',
    },
    startedAt: {
      description: 'When the import started',
      type: 'string',
      typeDescription: 'ISO 8601 datetime',
    },
    completedAt: {
      description: 'When the import finished, or null while it is still running',
      type: 'string | null',
      typeDescription: 'ISO 8601 datetime',
    },
  }}
/>
