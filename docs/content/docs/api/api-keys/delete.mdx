```
POST /api/auth/api-key/delete
```

Revokes (deletes) an existing API key. The key is identified by its `keyId`.

<Callout type="warn">This endpoint uses **POST**, not HTTP DELETE. Revoking a key is immediate and permanent — any client still using it will start receiving 401 responses.</Callout>

<Callout type="warn">This endpoint authenticates with your logged-in dashboard **session**, not an API key. Send your session cookie with the request.</Callout>

### Request Body

<TypeTable
  type={{
    keyId: {
      description: 'The id of the API key to delete',
      type: 'string',
      required: true,
    },
  }}
/>

### Response

Returns a success flag.

<TypeTable
  type={{
    success: {
      description: 'Indicates the API key was successfully deleted',
      type: 'boolean',
    },
  }}
/>
