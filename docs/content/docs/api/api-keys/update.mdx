```
POST /api/auth/api-key/update
```

<Callout>This endpoint uses **POST**, not PUT or PATCH.</Callout>

Updates mutable fields on an existing API key. The key is identified by its `keyId`.

<Callout type="warn">This endpoint authenticates with your logged-in dashboard **session**, not an API key. Send your session cookie with the request.</Callout>

### Request Body

<TypeTable
  type={{
    keyId: {
      description: 'The id of the API key to update',
      type: 'string',
      required: true,
    },
    name: {
      description: 'A new label for the key',
      type: 'string',
      required: false,
    },
    enabled: {
      description: 'Enable or disable the key',
      type: 'boolean',
      required: false,
    },
    expiresIn: {
      description: 'New expiry in seconds from now. Pass null to make the key never expire.',
      type: 'number | null',
      required: false,
    },
    rateLimitEnabled: {
      description: 'Enable or disable rate limiting on the key',
      type: 'boolean',
      required: false,
    },
  }}
/>

### Response

Returns the updated API key object (without the secret key value).

<TypeTable
  type={{
    id: {
      description: "The key's identifier",
      type: 'string',
    },
    name: {
      description: 'The label for the key',
      type: 'string | null',
    },
    start: {
      description: 'Non-secret leading characters of the key, for display',
      type: 'string | null',
    },
    prefix: {
      description: 'The key prefix',
      type: 'string | null',
    },
    userId: {
      description: 'ID of the user that owns the key',
      type: 'string',
    },
    enabled: {
      description: 'Whether the key is currently enabled',
      type: 'boolean',
    },
    expiresAt: {
      description: 'Expiration timestamp, or null if it never expires',
      type: 'string | null',
    },
    createdAt: {
      description: 'Creation timestamp',
      type: 'string',
    },
    updatedAt: {
      description: 'Last update timestamp',
      type: 'string',
    },
    rateLimitEnabled: {
      description: 'Whether rate limiting is enabled',
      type: 'boolean',
    },
    rateLimitMax: {
      description: 'Requests allowed per window',
      type: 'number | null',
    },
    rateLimitTimeWindow: {
      description: 'Window length in milliseconds',
      type: 'number | null',
    },
  }}
/>

**Note:** Additional fields may be present in the response object.
