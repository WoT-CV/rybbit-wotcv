```
PUT /api/organizations/:organizationId/teams/:teamId
```

Updates a team's name, members, or site access. Requires admin/owner role.

When `memberUserIds` or `siteIds` is provided, it **replaces** the team's entire existing set (it is not merged). Omit a field to leave it unchanged.

### Path Parameters

<TypeTable
  type={{
    organizationId: {
      description: 'Organization ID',
      type: 'string',
      required: true,
    },
    teamId: {
      description: 'ID of the team to update',
      type: 'string',
      required: true,
    },
  }}
/>

### Request Body

<TypeTable
  type={{
    name: {
      description: 'Updated team name',
      type: 'string',
      required: false,
    },
    memberUserIds: {
      description: 'User IDs to set as the team members (replaces the existing set)',
      type: 'string[]',
      required: false,
    },
    siteIds: {
      description: 'Site IDs to set as the team site access (replaces the existing set)',
      type: 'number[]',
      required: false,
    },
  }}
/>

### Response

<TypeTable
  type={{
    success: {
      description: 'Whether the team was updated',
      type: 'boolean',
    },
  }}
/>
