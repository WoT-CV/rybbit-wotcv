```
POST /api/organizations/:organizationId/teams
```

Creates a team in an organization. Requires admin/owner role.

### Path Parameters

<TypeTable
  type={{
    organizationId: {
      description: 'Organization ID',
      type: 'string',
      required: true,
    },
  }}
/>

### Request Body

<TypeTable
  type={{
    name: {
      description: 'Team name',
      type: 'string',
      required: true,
    },
    memberUserIds: {
      description: 'User IDs to add to the team (must be members of the organization)',
      type: 'string[]',
      required: false,
    },
    siteIds: {
      description: 'Site IDs the team can access (must belong to the organization)',
      type: 'number[]',
      required: false,
    },
  }}
/>

### Response

Returns the created team with HTTP 201.

<TypeTable
  type={{
    id: {
      description: 'Team ID',
      type: 'string',

      description: 'Team name',
      type: 'string',
    },

      description: 'Organization ID the team belongs to',
      type: 'string',
    },

      description: 'Timestamp when the team was created',

      description: 'Timestamp when the team was last updated',

      typeDescription: 'ISO 8601 datetime',
    },
    members: {
      description: 'Array of added user IDs',
      type: 'string[]',
    },
    siteIds: {

    },
  }}
/>
