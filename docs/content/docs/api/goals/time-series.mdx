```
GET /api/sites/:site/goals/time-series
```

Returns goal conversions and conversion rate over time.

### Path Parameters

<TypeTable
  type={{
    site: {
      description: 'Site ID or site identifier',
      type: 'string | number',
      required: true,
    },
  }}
/>

### Query Parameters

Accepts all [Common Parameters](/docs/api/getting-started#common-parameters) (time and filters) plus the following:

<TypeTable
  type={{
    goal_ids: {
      description: 'The goals to include',
      type: 'string',
      typeDescription: 'Comma-separated goal IDs (e.g. "1,2") or a JSON array. If omitted, the response is empty.',
      required: true,
    },
    bucket: {
      description: 'Time bucket size for grouping data',
      type: 'TimeBucket',
      typeDescription: '"minute" | "five_minutes" | "ten_minutes" | "fifteen_minutes" | "hour" | "day" | "week" | "month" | "year"',
      default: 'hour',
    },
  }}
/>

#### Bucket Selection Guide

| Bucket | Best For |
|--------|----------|
| `minute` | Real-time monitoring (last hour) |
| `five_minutes` | Short-term analysis (last few hours) |
| `hour` | Daily analysis |
| `day` | Weekly/monthly analysis |
| `week` | Monthly/quarterly analysis |
| `month` | Yearly analysis |

### Response

<TypeTable
  type={{
    time: {
      description: 'Start of the time bucket',
      type: 'string',
      typeDescription: 'ISO 8601 datetime',
    },
    goal_id: {
      description: 'ID of the goal',
      type: 'number',
    },
    conversions: {
      description: 'Number of distinct converting sessions in this bucket',
      type: 'number',
    },
    total_sessions: {
      description: 'Total number of sessions in this bucket',
      type: 'number',
    },
    conversion_rate: {
      description: 'Conversion rate as a fraction',
      type: 'number',
      typeDescription: 'Range: 0-1',
    },
  }}
/>
